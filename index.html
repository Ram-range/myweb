🚀 Belajar GitHub

Website ini dibuat untuk menjelaskan cara menggunakan GitHub secara sederhana.

1. Apa itu GitHub?

GitHub adalah tempat untuk menyimpan dan mengelola kode program secara online.

2. Cara Membuat Repository

  • Login ke GitHub
  • Klik "New Repository"
  • Isi nama project
  • Klik "Create Repository"

3. Upload File

  • Klik "Add file"
  • Pilih "Upload files"
  • Masukkan file HTML & CSS
  • Klik "Commit"

4. Publikasikan Website

  • Masuk ke Settings
  • Pilih menu "Pages"
  • Pilih branch "main"
  • Save
